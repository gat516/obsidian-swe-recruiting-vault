# technical workflow

## 1. prime (once per pattern)
- read labuladong chapter
- take notes with [[templates/labuladong-notes-template]] in `notes/`
- distill into the pattern hub's `## signal` + `## skeleton`

## 2. solve (25–30 min per problem)
- create problem note from [[templates/problem-template]] in `problems/`
- fill sections 1–4 (ticket), solo
- use ai, solve using claude helping you through logic → fill 5–7
- stuck → peek, then close and re-implement in section 5

## 3. consolidate
- update pattern hub
- update `flashcards/pattern.md`
- bump `attempts +1`
- flip `#wip` → `#done`
- [[07_daily-log]]

## weekly
- re-read touched pattern hubs
- review `flashcards/pattern.md` (tagged `#flashcard`)
